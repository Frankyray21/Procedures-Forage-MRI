Déposer la dernière version d'une procédure

Chaque fiche du site affiche automatiquement le PDF officiel. Pour mettre une procédure à jour, il suffit de remplacer son PDF.

  1. Trouvez la procédure ci-dessous et choisissez votre nouveau PDF.
  2. Cliquez « Télécharger renommé » : le fichier est renommé au bon nom automatiquement.
  3. Cliquez « Téléverser sur GitHub », glissez le fichier renommé, puis Commit changes.
  4. La fiche du site affichera la nouvelle version après quelques minutes (rechargez la page).
